# Baby Layette UX

Use esta skill quando mexer na experiência do enxoval.

Objetivo:
- O app deve ser fácil para uma mãe usar sem explicação longa.
- O fluxo principal é: ver o que falta, marcar comprado, ver presente, acompanhar gasto.
- Evitar excesso de campos na tela principal.
- Separar o que é essencial, comprado, pendente e presenteado.
- Deixar claro o que foi comprado por Papai, Mamãe ou presenteado por convidado.

Checklist obrigatório:
- Linguagem simples e familiar.
- Título do app: Enxoval do Timóteo.
- Selo visual para presente.
- Status claros: pendente, comprado, presenteado.
- Dashboard útil para Papai/Mamãe.
- Tela mais simples para convidado.
