# Mobile PWA Design

Use esta skill quando for melhorar interface mobile/PWA.

Objetivo:
- Deixar o app bonito, simples e confortável no celular.
- Priorizar uso real por Papai, Mamãe e convidados.
- Evitar aparência de planilha.
- Usar cards, botões grandes, espaçamento bom e contraste forte.
- Manter tema claro fixo, sem quebrar no modo escuro do iPhone.
- Preferir visual suave: azul bebê, off-white, bege suave, verde claro para comprado/presente.

Checklist obrigatório:
- Testar responsividade mobile.
- Evitar campos pequenos demais.
- Evitar botões perigosos expostos, como excluir direto.
- Priorizar fluxo de uso simples.
- Melhorar legibilidade para quem não é técnico.
