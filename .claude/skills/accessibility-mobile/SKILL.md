# Accessibility Mobile

Use esta skill quando alterar interface, botões, formulários e cores.

Objetivo:
- Garantir que o app seja confortável no celular.
- Botões devem ser grandes o suficiente para toque.
- Texto deve ter contraste adequado.
- Não depender só de cor para indicar status.
- Inputs devem ter labels claros.
- Modais devem ser fáceis de fechar e entender.

Checklist obrigatório:
- Botões principais com área de toque confortável.
- Labels visíveis.
- Texto legível no celular.
- Estados de erro claros.
- Confirmação antes de ações destrutivas.
- Tema claro preservado mesmo com iPhone em modo escuro.
